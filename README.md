# Loquitir AI

High-performance AI receptionist solutions for personal injury law firms.

## Development

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```
   npm install
   ```
2. Run the dev server:
   ```
   npm run dev
   ```

## Build

```
npm run build
```

The production build will be output to the `dist/` directory.

## Deploy

This site is configured for static hosting. Upload the contents of `dist/` to your hosting provider (e.g., Hostinger).
